## üåü Key Features

### üß† Customizable Explanation Levels (The "Explain Like..." Feature)

A core feature of Explainify is the ability for users to precisely control the complexity of the output. Instead of a one-size-fits-all explanation, you can select from a range of predefined levels, giving you the perfect starting point for learning.

| Level | Description | Target Audience |
| :--- | :--- | :--- |
| **ELI5** | **E**xplain **L**ike **I**'m **5** | Simple language, analogies, and basic concepts. Avoids technical jargon. |
| **ELITeen** | **E**xplain **L**ike **I**'m a **Teen** | Uses clear, modern language. Introduces core terminology without getting lost in detail. |
| **EL18** | **E**xplain **L**ike **I**'m an **18**-year-old | Explanations suitable for a college student. Uses necessary technical terms, assuming basic subject knowledge. |
| **Expert** | **Expert** Mode | A detailed, technical breakdown with minimal simplification. Perfect for quick reviews or deep dives. |

### üìÖ Daily New Facts

Keep your mind sharp and discover something new every day! The **Daily New Fact** feature provides a bite-sized, interesting explanation of a random complex topic.

* **Fresh Content:** A new, curated fact is generated and displayed on the homepage daily.
* **Quick Learning:** Perfect for a 30-second burst of knowledge.
* **Contextual Links:** Each fact links directly to the main explanation engine, allowing you to instantly request an **ELI5** or **EL18** breakdown of the day's topic.

### Other Features
* **Intuitive Interface:** A clean, minimal design focused on readability.
* **Topic Search:** Quickly find explanations for thousands of subjects.
